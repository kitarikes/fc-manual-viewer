前置き：この資料のスコープ
マニュアル制作には2つの側面がある。

- ① 動画制作・入稿側：撮影・編集・アップロードのフロー（吉良さん検討）
- ② システム運用側：スタッフへの見せ方・権限管理・マニュアル閲覧のしやすさ

この資料では ② システム運用側 に絞って比較する。

1. 課題の整理
- 現状：Teachme Bizを使っているが、システム運用の観点で問題がある
  - 年間約100万円のランニングコストが永続的にかかる
  - 権限管理・カスタマイズなど、FC展開に必要な機能が不足
- ゴール：FC展開に耐えうる、費用対効果の高い、内製運用可能なマニュアルシステム

2. 選択肢の比較

Teach me Biz 継続
- メリット
  - 初期コストが低い（契約継続のみ）
  - 素人でもすぐ使える、更新・運用が簡単


- デメリット
  - 年間100万円のランニングコストが永続的にかかる
  - 権限管理・カスタマイズがほぼできない
  - FC向けに提供するには品質・信頼性が不足

自社開発
- メリット
  - カスタマイズ自由（PDF添付・多言語・権限管理など要望を全部実装できる）
  - 長期的にランニングコストが下がる
  - FC向けのプロ仕様に作り込める → 商品としての価値が高い
  - 将来の機能追加・多言語展開にも対応しやすい
  
- デメリット
  - 初期コストがかかる（150万〜200万円）
  - 開発期間が必要

3. 推薦案
- フェーズ1（〜7月）: Teachme Biz契約を最大活用 → マニュアル素材を量産
- フェーズ2（7月〜）: 自社システムへコンテンツ移行
- 理由：二重コストを避けながらスムーズに移行できる

4. 費用感
- 動画制作費：本数・範囲が確定次第見積もり
- システム開発費：150万〜200万円（要件定義後に確定）
- プラン提示方法：松竹梅の3段階で選択肢を提示（今後検討していく）
