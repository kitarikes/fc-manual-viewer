# 味和商店 マニュアル閲覧システム 提案資料

クライアント（吉良さん）向けの読み合わせ会用ビジュアル資料。

## 画面構成

- `index.html` — 資料トップ（概要 + 各資料へのリンク）
- `requirements.html` — 要求整理（フェーズ別ユースケース一覧）
- `schedule.html` — プロジェクトスケジュール（ガントチャート + スプリント計画）
- `estimate.html` — 見積もり（フェーズ別支払い + 支払いスケジュール）

## 目的

MDファイルでは読みにくい提案資料を、視覚的に整理してクライアントとの合意形成をスムーズにする。
